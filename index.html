¿Qué es?

Una librería JavaScript que facilita:

  • Manipular elementos del DOM
  • Gestionar eventos
  • Usar AJAX

Manipular elementos del DOM

Usando JavaScript estándar: 

                      
                        document.getElementById("nombre") = "Germán";

Usando jQuery: 

                      
                        $("#nombre").val("Germán);
